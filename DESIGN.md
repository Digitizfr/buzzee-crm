# Design System Document

## Theme Overview

This document outlines the core visual principles and token values that define our design system.

### Color Palette

Our brand utilizes a specific color palette to ensure consistency and brand recognition across all interfaces.

* **Primary Color:** `#2E5748` - The brand's most distinctive chromatic color, suitable for buttons, CTAs, and key interactive elements.
* **Secondary Color:** `#CEFEAB` - A supporting color for less prominent UI elements, chips, and secondary actions.
* **Tertiary Color:** `#1E3B33` - An additional accent color for highlights, badges, or decorative elements.
* **Neutral Color:** `#F3F3F4` - A neutral base color for backgrounds, surfaces, and non-chromatic elements.

The current color mode is **light**.

### Typography

Our typography system is designed for readability and hierarchy, utilizing distinct fonts for headlines, body text, and labels.

* **Headline Font:** `manrope` - Used for titles and prominent text, conveying brand voice.
* **Body Font:** `inter` - Applied to general text, ensuring legibility for longer content.
* **Label Font:** `inter` - Employed for UI labels, buttons, and smaller annotations.

### Shape and Form

The visual characteristics of our UI components are defined by their roundedness.

* **Roundedness:** `2` (Moderate) - This setting provides a balanced, friendly feel to our UI elements.

### Spacing

Our spacing strategy ensures clarity and hierarchy, making interfaces easy to scan and interact with.

* **Spacing:** `2` (Normal) - This provides a balanced amount of whitespace, making components feel organized and legible.
